# BMI Calculator

## Student

Dhruv Tandel

## GitHub

https://github.com/Dhruv56mi/bmi-calculator

## Features

- Enter Age
- Select Gender
- Enter Height (cm)
- Enter Weight (kg)
- Calculate BMI
- Display BMI category

## BMI Formula

BMI = Weight (kg) / Height (m)^2

## Technologies Used
- Flutter
- Dart
## Run the Project

----flutter pub get
----flutter run

## Docker Commands

### Pull Image
docker pull username/bmi-calculator

### Run Image
docker run username/bmi-calculator

### Create Container
docker create username/bmi-calculator

### Stop Container
docker stop CONTAINER_ID
